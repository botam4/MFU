## English

This is a DIY project to develop the MFU filament management unit that helps upgrade 3D printers to multi-color printing. The first version supports 4 colors, with future updates expanding up to 24 colors.

### Getting Started

1. Prepare components according to the list in [BOM_linh_kien.md](BOM_linh_kien.md)
2. **Note:** When selecting stepper motor drivers, choose models that support 1/4 microstepping configuration
